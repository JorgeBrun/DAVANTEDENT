# DAVANTEDENT

Aplicación web para la gestión de citas de una clínica dental ficticia llamada DavanteDent.

## Descripción
Este proyecto ha sido desarrollado como trabajo de enfoque del módulo
**Desarrollo Web en Entorno Cliente**.  
Permite gestionar citas de pacientes mediante un formulario web, validando
los datos introducidos y almacenando la información en cookies del navegador.

## Funcionalidades
- Alta de citas
- Edición de citas
- Eliminación de citas
- Validación de datos:
  - Teléfono (9 dígitos)
  - DNI (8 números y letra mayúscula)
  - Pacientes mayores de edad
- Persistencia de datos mediante cookies

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Cookies del navegador

## Ejecución
Abrir el archivo `src/index.html` en un navegador web.
